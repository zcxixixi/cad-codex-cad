# 计算校核说明

## 结论

报告采用的水平电缆计算方法与参考PDF一致：

```text
C = ((F + N) / 2 + L) × n × (1 + 10%)
```

其中 `L=2 m`，`n=TP+TD+AP`；每层、每配线区域分别计算，箱数按
`ceil(C/305)` 向上取整。

## 参考PDF中的错误

参考PDF的公式和计算思路可采用，但部分汇总及表格算术错误，不能直接照抄：

- 1号楼北区逐行TP合计为5，不是汇总栏的4。
- 1号楼三部分数据点合计为218，语音点为7。
- 全部数据点为460，语音点为38，总信息点为498。
- 参考PDF部分水平电缆长度不满足其自身公式。
- 参考PDF材料表中的120箱、472个数据点、37个语音点与逐行数据不一致。

参考距离中1号教学楼有9个配线区域的最远距离超过90 m。当前电缆数量按原
距离保守计算；施工深化必须调整FD位置、增设FD或优化路由，使永久链路不超过
90 m。
