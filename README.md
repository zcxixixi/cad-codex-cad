# 建筑物信息设施系统课程设计

某中学1号教学楼、2号教学楼和行政综合楼的信息网络、综合布线、有线电视及公共广播系统课程设计。

## 交付内容

- `outputs/建筑物信息设施系统课程设计报告.pdf`：完整报告，末尾含18页CAD图纸附录。
- `outputs/建筑物信息设施系统课程设计报告.tex`：LaTeX源文件。
- `outputs/课程设计封面及评分表.pdf`：报告使用的学校封面和评分表。
- `outputs/附录_CAD图纸.pdf`：独立图纸附录。
- `计算校核说明.md`：与参考PDF逐项对照后的独立复算结论。
- `outputs/01A`至`03A`：保留天正代理对象的完整DWG，提交和打印优先使用。
- `outputs/01`至`03`：AutoCAD 2000兼容DWG。
- `outputs/04_信息网络_综合布线_有线电视_公共广播系统图_T3.dwg`：四系统系统图。

## 核算结果

- 普通数据点：288
- 无线AP点：172
- 数据点合计：460
- 语音点：38
- 信息点总计：498
- Cat6A水平电缆计算长度：23.377 km
- 按配线区域逐层取整：93箱

## 我如何用 Codex 完成

1. 我把任务书、示例报告、建筑底图和学校模板交给 Codex，让它先读取文件并梳理全部交付要求。
2. Codex 从参考资料中提取TP、TD、AP点位以及距离数据，按建筑、楼层和区域重新计算，检查分项、合计和电缆箱数。
3. 我安装并配置了CAD转换与查看环境，Codex 使用脚本处理DWG/DXF、整理系统图，并把完整图纸输出为PDF附录。
4. Codex 按参考计算方法完成水平电缆、设备容量和材料数量计算，同时检查图纸点位与报告统计是否一致。
5. Codex 使用LaTeX生成课程设计报告，把计算表、系统拓扑、设备材料表和CAD图纸附录统一排版。
6. 我提供学校封面和评分表，Codex 将其替换到报告前两页，重新编译并检查最终PDF的页数、内容和版式。
7. 最后，Codex 整理源文件、报告、CAD图纸和校核说明，提交到本GitHub仓库。

## 编译

```bash
cd outputs
latexmk -xelatex -interaction=nonstopmode -halt-on-error 建筑物信息设施系统课程设计报告.tex
```

首次打开完整DWG时，如软件提示代理图形，应选择显示代理图形。
